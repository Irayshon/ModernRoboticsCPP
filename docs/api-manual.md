## Generate Locally

From the repository root:

```sh
doxygen docs/Doxyfile
```

Open the generated HTML:

`docs/doxygen/html/index.html`

## Notes

- The API uses Eigen types such as `Eigen::MatrixXd`, `Eigen::VectorXd`, and `Eigen::Vector3d`.
- All public functions are documented with parameter types and expected meanings.
